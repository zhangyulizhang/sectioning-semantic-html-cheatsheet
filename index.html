Sectioning Semantic HTML Cheatsheet

Tag Name HTML tag Explanation
Header <header> Used to add a header to a document
Nav <header> Used to add navigation links to a webpage.
Footer <footer> Used to add a footer to the document
Main <main> Used to specify the main content in the document.
Aside <aside> Used to add content that is not essential to understanding the main content
Article <article> Used to add an independent self-contained block of content.
Section <section> Used to add a stand-alone section, usually used within the body and article elements.
Details <details> Used to add a collapsed section that can be expanded if the user wishes to view it.
Summary <summary> Used to summarise or add a caption to a <details> element.
Heading <h1>, ... , <h6> Used to add headings on a document, <h1> indicates the most importand one and <h1> the least.
